## 提示框（Alert）
提示框.

## API

### 属性
属性 | 说明 | 类型 | 可选值 | 默认值
---|---|---|---|---
visible | 是否显示, v-model | `Boolean` | - | `true`
type |  警告提示样式	 | `String` | (`info` `success` `error` `warning`) | `info` 
title |  标题 | `String` | - | - 
desc |  辅助性文字, 也可通过默认 slot 传入 | `String` | - | - 
icon |  图表icon | `String` | - | -
showIcon |  是否显示图标 | `Boolean` | - | `true`
closable |  是否可关闭 | `Boolean` | - | `false`
